# App-de-Movilidad-Incluyente-y-Turismo-local-en-STPM
CaliMIO: App bilingue que emplea información pública del Masivo Integrado de Occidente MIO de Cali, Colombia (movilidad y turismo incluyente en transporte público). Se utiliza la extensión cn.kevinkun.RegEx.aix. App Sujeta a Modificaciones Futuras. Desarrollada por el equipo Angi Pardo-Steven Esquea de Mente Irreverente. Las imágenes de la guía turística son ilustrativas (de otros autores), temporales y cambiantes. Imagen del bus (de uso libre) tomada de Canva.
